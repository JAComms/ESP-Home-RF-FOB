# ESP-Home-RF-FOB

YAML code for ESP-Home to Transmit & Receive 433Mhz Key Fob Data.

I wrote this small code to program custom FOB data into a few cheap Ebay FOBS


Tested on ESP32-S3-WROOM-1U & ESP32-WROOM-32D
